# Style Guide

## Colors

- Primary: `#5a1cbb`
- Accent: `#ff6a00`
- Neutral Background: `white`
- Nav Background: `#ff45ff`
- Footer Background: `rgb(4, 53, 53)`
- Header Background: `#5a1cbb` (matches primary colour)
- Header Text: `white`
- Link: `navy`
- Link Hover: `#ff6a00`

## Typography

- Font Family: Arial, Helvetica, sans-serif
- Base Font Size: 18px
- Heading Font Size: 2rem
- Paragraph Font Size: 1.2rem

## Layout

- Mobile-first design
- Breakpoint at 768px for desktop adjustments
- Flexbox for image containers and navigation

## Other

- Sticky navigation bar and skip link enhance keyboard accessibility
- Light/dark theme support using CSS variables and a toggle button
