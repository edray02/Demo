# Building A Bridge

A simple webpage guide on building a bridge, built with HTML and CSS.

## Overview

This project provides a basic guide to building a bridge, including introduction, materials, steps, and conclusion. It's designed to be responsive and accessible.

## Built With

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- Mobile-first responsive design

## Features

- Responsive layout for mobile and desktop
- Clean, semantic HTML structure
- CSS variables for easy theming
- Accessible navigation and content
- Sticky navigation and skip link for keyboard users
- Active section highlighting in the menu
- Light/dark theme toggle

## Getting Started

1. Clone or download the project.
2. Open `index.html` in your browser.
